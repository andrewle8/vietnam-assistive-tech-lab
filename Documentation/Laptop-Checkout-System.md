# Laptop Checkout System

## Tổng quan / Overview

**Mục đích:** Theo dõi việc mượn/trả máy tính xách tay để đảm bảo trách nhiệm quản lý thiết bị.

**Purpose:** Track laptop loans for accountability — know which student has which laptop at all times.

**Thành phần / Components:**
- **Google Form** — biểu mẫu nhập liệu trên điện thoại (staff-operated, sighted staff use their phone)
- **Google Sheet** — bảng tính tự động với dashboard trạng thái (auto-populated with a live status dashboard)
- **Sổ giấy / Paper backup** — bản ghi giấy dự phòng khi không có internet

**Quy trình / Workflow:**
- Nhân viên (sighted staff) quét biểu mẫu Google Form trên điện thoại khi học sinh mượn hoặc trả máy
- Mã học sinh (student identifier) theo định dạng **STU-XXX** (ví dụ: STU-001, STU-042)
- Tên học sinh được ghi nhận tại thời điểm mượn máy (student names are captured at checkout)

## Google Form Setup

Step-by-step instructions to create the checkout form in Google Forms.

### 1. Create the form

1. Go to [forms.google.com](https://forms.google.com)
2. Click **Blank form** (biểu mẫu trống) to create a new form

### 2. Set form title and description

- **Title:** `Hệ Thống Mượn Máy Tính - Vietnam Lab`
- **Description:** `Biểu mẫu theo dõi mượn/trả máy tính xách tay`

### 3. Add form fields

**Field 1 — Hành động (Action)**
- Type: **Dropdown**
- Required: Yes
- Options:
  - `Mượn máy`
  - `Trả máy`

**Field 2 — Số máy tính (Laptop Number)**
- Type: **Dropdown**
- Required: Yes
- Options:
  - `PC-01`
  - `PC-02`
  - `PC-03`
  - `PC-04`
  - `PC-05`
  - `PC-06`
  - `PC-07`
  - `PC-08`
  - `PC-09`
  - `PC-10`
  - `PC-11`
  - `PC-12`
  - `PC-13`
  - `PC-14`
  - `PC-15`
  - `PC-16`
  - `PC-17`
  - `PC-18`
  - `PC-19`

**Field 3 — Mã học sinh (Student ID)**
- Type: **Short answer**
- Required: Yes
- Response validation:
  - Type: Regular expression
  - Pattern: `STU-\d{3}`
  - Custom error text: `Vui lòng nhập đúng định dạng STU-XXX (ví dụ: STU-001)`

**Field 4 — Tên học sinh (Student Name)**
- Type: **Short answer**
- Required: Yes

**Field 5 — Ghi chú (Notes)**
- Type: **Paragraph**
- Required: No (optional)

### 4. Link to Google Sheet

1. Click the **Responses** tab at the top of the form
2. Click the green **Google Sheets** icon (Link to Sheets)
3. Select **Create a new spreadsheet**
4. The form responses will now auto-populate in the linked sheet

## Dashboard Tab Setup

Set up a live status dashboard in the linked Google Sheet so staff can see at a glance which laptops are currently checked out.

### 1. Rename the responses tab

In the linked Google Sheet, rename the auto-created tab (usually called "Form Responses 1") to:

```
Phản hồi
```

### 2. Confirm column layout

The `Phản hồi` tab should have these columns (auto-generated by the form):

| Column | Header (Vietnamese) | Content |
|--------|-------------------|---------|
| A | Dấu thời gian | Timestamp |
| B | Hành động | Action (Mượn máy / Trả máy) |
| C | Số máy tính | Laptop number (PC-01 through PC-19) |
| D | Mã học sinh | Student ID (STU-XXX) |
| E | Tên học sinh | Student name |
| F | Ghi chú | Notes |

### 3. Create the dashboard tab

Create a new tab called:

```
Bảng theo dõi
```

### 4. Set up headers (Row 1)

| Column | Header |
|--------|--------|
| A | Máy tính |
| B | Trạng thái |
| C | Mã học sinh |
| D | Tên học sinh |
| E | Ngày mượn |
| F | Ghi chú |

### 5. Populate Column A (Laptop names)

Enter PC names in cells A2 through A20 (hardcoded):

| Cell | Value |
|------|-------|
| A2 | PC-01 |
| A3 | PC-02 |
| A4 | PC-03 |
| A5 | PC-04 |
| A6 | PC-05 |
| A7 | PC-06 |
| A8 | PC-07 |
| A9 | PC-08 |
| A10 | PC-09 |
| A11 | PC-10 |
| A12 | PC-11 |
| A13 | PC-12 |
| A14 | PC-13 |
| A15 | PC-14 |
| A16 | PC-15 |
| A17 | PC-16 |
| A18 | PC-17 |
| A19 | PC-18 |
| A20 | PC-19 |

### 6. Column B — Trạng thái (Status)

Enter this formula in cell **B2**, then drag down to **B20**:

```
=IF(COUNTIF('Phản hồi'!C:C,A2)=0,"Sẵn sàng",IF(INDEX(SORT(FILTER('Phản hồi'!B:B,'Phản hồi'!C:C=A2),FILTER('Phản hồi'!A:A,'Phản hồi'!C:C=A2),FALSE),1,1)="Mượn máy","Đang mượn","Sẵn sàng"))
```

This looks up the most recent action for each laptop. If no records exist or the last action was "Trả máy" (returned), status shows **Sẵn sàng** (Available). If the last action was "Mượn máy" (checked out), status shows **Đang mượn** (Checked Out).

### 7. Column C — Mã học sinh (Student ID)

Enter this formula in cell **C2**, then drag down to **C20**:

```
=IF(B2="Đang mượn",INDEX(SORT(FILTER('Phản hồi'!D:D,'Phản hồi'!C:C=A2),FILTER('Phản hồi'!A:A,'Phản hồi'!C:C=A2),FALSE),1,1),"—")
```

Shows the student ID of the current borrower, or "—" if available.

### 8. Column D — Tên học sinh (Student Name)

Enter this formula in cell **D2**, then drag down to **D20**:

```
=IF(B2="Đang mượn",INDEX(SORT(FILTER('Phản hồi'!E:E,'Phản hồi'!C:C=A2),FILTER('Phản hồi'!A:A,'Phản hồi'!C:C=A2),FALSE),1,1),"—")
```

Shows the student name of the current borrower, or "—" if available.

### 9. Column E — Ngày mượn (Checkout Date)

Enter this formula in cell **E2**, then drag down to **E20**:

```
=IF(B2="Đang mượn",TEXT(INDEX(SORT(FILTER('Phản hồi'!A:A,'Phản hồi'!C:C=A2),FILTER('Phản hồi'!A:A,'Phản hồi'!C:C=A2),FALSE),1,1),"DD/MM/YYYY"),"—")
```

Shows the checkout date in DD/MM/YYYY format, or "—" if available.

### 10. Column F — Ghi chú (Notes)

Enter this formula in cell **F2**, then drag down to **F20**:

```
=IF(B2="Đang mượn",INDEX(SORT(FILTER('Phản hồi'!F:F,'Phản hồi'!C:C=A2),FILTER('Phản hồi'!A:A,'Phản hồi'!C:C=A2),FALSE),1,1),"—")
```

Shows any notes from the most recent checkout, or "—" if available.

### 11. Conditional formatting

Highlight checked-out laptops so they stand out visually:

1. Select the range **A2:F20**
2. Go to **Format > Conditional formatting**
3. Set "Format cells if": **Custom formula is**
4. Enter formula: `=$B2="Đang mượn"`
5. Set fill color to **light red** (e.g., `#F4C7C3`)
6. Click **Done**

Rows where a laptop is currently checked out will be highlighted in light red.

## Hướng dẫn sử dụng / Usage Instructions

### Khi học sinh mượn máy (When a student borrows a laptop)

1. Mở biểu mẫu Google Form trên điện thoại
   (Open the Google Form on your phone)
2. Chọn **Mượn máy** trong mục "Hành động"
   (Select **Mượn máy** under "Hành động")
3. Chọn số máy tính (ví dụ: PC-05)
   (Select the laptop number, e.g., PC-05)
4. Nhập mã học sinh (ví dụ: STU-001)
   (Enter the student ID, e.g., STU-001)
5. Nhập tên học sinh
   (Enter the student's name)
6. Gửi biểu mẫu
   (Submit the form)

### Khi học sinh trả máy (When a student returns a laptop)

1. Mở biểu mẫu Google Form trên điện thoại
   (Open the Google Form on your phone)
2. Chọn **Trả máy** trong mục "Hành động"
   (Select **Trả máy** under "Hành động")
3. Chọn số máy tính đang trả
   (Select the laptop number being returned)
4. Nhập mã học sinh và tên
   (Enter the student ID and name)
5. Gửi biểu mẫu
   (Submit the form)

### Không có internet (No internet available)

1. Ghi thông tin vào sổ giấy trên tường
   (Write the information on the paper log on the wall)
2. Khi có internet, nhập lại vào Google Form
   (When internet is available, enter the data into the Google Form)

### Kiểm tra máy đang mượn (Check which laptops are currently out)

1. Mở Google Sheet đã liên kết
   (Open the linked Google Sheet)
2. Chọn tab **Bảng theo dõi**
   (Select the **Bảng theo dõi** tab)
3. Các máy có trạng thái **Đang mượn** (highlighted in red) đang được mượn
   (Laptops with status **Đang mượn** (highlighted in red) are currently checked out)
