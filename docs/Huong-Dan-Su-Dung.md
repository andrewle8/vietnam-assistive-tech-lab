# Huong Dan Su Dung May Tinh / Laptop User Guide

Tai lieu nay huong dan cach su dung may tinh voi NVDA.
This guide explains how to use the laptop with NVDA screen reader.

---

## Bat Dau / Getting Started

Khi may tinh bat len, NVDA se tu dong chay va noi "NVDA dang chay".
When the laptop turns on, NVDA starts automatically and says "NVDA is running".

Ban se nghe mot loi chao. Sau do, ban o man hinh Desktop.
You will hear a welcome message. After that, you are on the Desktop.

---

## Dieu Huong Tren Desktop / Desktop Navigation

Desktop co 21 bieu tuong, duoc danh so tu 01 den 21.
The desktop has 21 icons, numbered 01 to 21.

Dung phim mui ten (Trai, Phai, Len, Xuong) de di chuyen giua cac bieu tuong.
Use arrow keys (Left, Right, Up, Down) to move between icons.

Nhan Enter de mo chuong trinh.
Press Enter to open a program.

## Phim Tat NVDA Quan Trong / Important NVDA Shortcuts

Phim NVDA la phim Insert (hoac Caps Lock).
The NVDA key is Insert (or Caps Lock).

| Phim / Key | Chuc nang / Function |
|---|---|
| Insert + T | Nghe tieu de cua so / Read window title |
| Insert + B | Doc toan bo cua so / Read entire window |
| Insert + F7 | Danh sach lien ket / List of links |
| Insert + Space | Chuyen che do duyet / Toggle browse mode |
| Insert + N | Mo menu NVDA / Open NVDA menu |
| Insert + Q | Thoat NVDA / Quit NVDA |
| Ctrl | Dung doc / Stop reading |
| Insert + Mui ten xuong | Doc tu vi tri hien tai / Read from current position |
| Tab | Di chuyen den muc tiep theo / Move to next item |
| Shift + Tab | Di chuyen den muc truoc / Move to previous item |
| Alt + F4 | Dong chuong trinh / Close program |
| Alt + Tab | Chuyen chuong trinh / Switch programs |

## Go Tieng Viet / Typing Vietnamese

UniKey tu dong chay khi bat may.
UniKey starts automatically when the laptop turns on.

Mac dinh dung kieu go Telex.
Default input method is Telex.

Vi du Telex / Telex examples:
- aa = a (a mu)
- aw = a (a ngan)
- oo = o (o mu)
- ow = o (o moc)
- uw = u (u moc)
- dd = d (d ngang)
- s = dau sac, f = dau huyen, r = dau hoi, x = dau nga, j = dau nang

## Khi NVDA Bi Loi / If NVDA Breaks

Neu NVDA khong noi hoac noi sai:
If NVDA stops talking or speaks incorrectly:

1. Dung phim mui ten den bieu tuong so 21 "Khoi Phuc NVDA"
   Use arrow keys to icon 21 "Khoi Phuc NVDA"
2. Nhan Enter
   Press Enter
3. Doi vai giay, NVDA se khoi dong lai voi cai dat goc
   Wait a few seconds, NVDA will restart with default settings

---

## Dieu Chinh Am Luong / Volume Control

Am luong duoc dat o 70% moi khi bat may.
Volume is set to 70% each time the laptop starts.

Dung phim am luong tren ban phim de tang hoac giam.
Use the volume keys on the keyboard to increase or decrease.
