# Pre-Deployment Checklist for Vietnam Lab

Use this checklist to ensure you're fully prepared before traveling to Vietnam.

## 2-3 Weeks Before Deployment

### Software & USB Kit
- [ ] All installer files downloaded (see `Installers/README.md`)
- [ ] NVDA 2025.3.2+ downloaded
- [ ] Sao Mai VNVoice + Typing Tutor downloaded
- [ ] LibreOffice 24.8 LTS MSI downloaded
- [ ] Firefox ESR 128+ downloaded
- [ ] VLC downloaded
- [ ] VLC NVDA add-on downloaded (for VLC accessibility)
- [ ] LEAP Games downloaded (Tic-Tac-Toe, Tennis, Curve)
- [ ] All files copied to USB drives (create 3 identical drives)
- [ ] PowerShell scripts tested on one PC
- [ ] NVDA configuration file verified
- [ ] Test complete offline deployment on one PC

### Hardware Preparation
- [ ] All 10 Windows 11 PCs fully updated
- [ ] Windows updates completed (before disabling internet)
- [ ] Manufacturer bloatware removed
- [ ] User accounts created (document passwords)
- [ ] Power settings configured (never sleep)
- [ ] Windows Defender configured for offline mode
- [ ] All 10 Orbit Reader 20 devices charged
- [ ] Orbit Reader firmware updated to latest
- [ ] USB cables tested with each Orbit Reader
- [ ] 10 wired headphones purchased (over-ear, 3.5mm jack)
- [ ] USB extension cables (10x, for flexible positioning)
- [ ] Label maker with extra label tape

### Documentation
- [ ] Training materials received from Sao Mai (Vietnamese)
- [ ] All documentation translated to Vietnamese
- [ ] Troubleshooting guides printed (2 copies)
- [ ] Quick reference cards printed (10+ copies)
- [ ] Deployment plan printed
- [ ] Contact list printed (emergency numbers)

### Coordination with Sao Mai Center
- [ ] Confirmed Sao Mai staff availability during deployment
- [ ] Scheduled daily check-in times
- [ ] Received Vietnamese training materials
- [ ] Established communication channels (WhatsApp/Signal/Email)
- [ ] Confirmed post-deployment support plan
- [ ] Shared trip itinerary with Sao Mai

### Backup & Contingency
- [ ] 3 USB drives created (primary + 2 backups)
- [ ] Tested on multiple PCs to verify compatibility
- [ ] Laptop + charger packed (for troubleshooting)
- [ ] Extra USB cables and adapters packed
- [ ] Printed documentation in waterproof folder
- [ ] Digital backup stored in cloud (without installer files)

## 1 Week Before Departure

### Final Testing
- [ ] Full deployment rehearsal on one PC (time it!)
- [ ] Verify scripts complete in under 30 minutes per PC
- [ ] Test NVDA speech with VNVoice
- [ ] Test Orbit Reader 20 braille output
- [ ] Test LibreOffice with NVDA
- [ ] Test typing tutor launches and speaks
- [ ] Verify all software works 100% offline
- [ ] Document any issues and solutions

### Travel Preparation
- [ ] Flight tickets confirmed
- [ ] Accommodation booked near orphanage/deployment site
- [ ] Visa requirements met (if applicable)
- [ ] Travel insurance purchased
- [ ] Emergency contacts list created
- [ ] Power adapters for Vietnam (Type A/C, 220V)
- [ ] Backup power bank for phone
- [ ] Physical address of deployment site saved offline

### Packing List - Equipment
- [ ] 3x USB drives (deployment kit)
- [ ] Laptop + charger
- [ ] Label maker + labels
- [ ] USB cables (extras)
- [ ] USB extension cables (10x)
- [ ] Headphones for testing
- [ ] Small toolkit (screwdrivers, etc.)
- [ ] Cable ties/organizers
- [ ] Ethernet cables (backup for troubleshooting)
- [ ] External mouse (backup)

### Packing List - Documentation
- [ ] Printed deployment plan
- [ ] Printed scripts README (English + Vietnamese)
- [ ] Printed troubleshooting guides (Vietnamese)
- [ ] Quick reference cards (10+ copies)
- [ ] Contact list with emergency numbers
- [ ] Notebook + pens for documentation
- [ ] Business cards (for networking)

### Communication Setup
- [ ] International phone plan activated OR local SIM purchased
- [ ] WhatsApp/Signal installed and tested
- [ ] Sao Mai contacts added to phone
- [ ] Emergency contacts shared with family/team
- [ ] Calendar reminders for check-ins
- [ ] Backup communication method established

## Day Before Travel

### Final Verification
- [ ] All USB drives tested one last time
- [ ] Laptop charged and working
- [ ] All equipment packed and ready
- [ ] Printed materials organized in binder
- [ ] Confirmed arrival time with orphanage/Sao Mai
- [ ] Shared flight details with local contacts
- [ ] Reviewed deployment timeline
- [ ] Sleep well! 😊

### Digital Backup
- [ ] Copy USB kit contents to laptop (backup)
- [ ] Save contact list to phone
- [ ] Download offline maps for navigation
- [ ] Screenshot important information
- [ ] Charge all devices fully

## On Arrival in Vietnam

### First 24 Hours
- [ ] Contact Sao Mai to confirm arrival
- [ ] Visit deployment site and inspect facilities
- [ ] Verify power outlets and internet (if available)
- [ ] Meet orphanage staff and introduce yourself
- [ ] Unpack equipment and organize workspace
- [ ] Test one PC with USB kit to verify everything works
- [ ] Address any unexpected issues before Day 1

## Equipment Inventory

### For Each of 10 Stations
- [ ] Windows 11 PC (already on-site or shipped)
- [ ] Orbit Reader 20 braille display
- [ ] Wired headphones
- [ ] USB cable for Orbit Reader
- [ ] USB extension cable (optional)
- [ ] Label (PC-1 through PC-10)

### Shared Equipment
- [ ] 3x USB deployment kits
- [ ] 1x laptop for troubleshooting
- [ ] Label maker
- [ ] Extra cables and adapters
- [ ] Printed documentation binder
- [ ] Toolkit

## Success Criteria

Before leaving the deployment site:
- [ ] All 10 PCs boot and run NVDA automatically
- [ ] All 10 Orbit Readers connect and display braille
- [ ] Speech output works in Vietnamese on all stations
- [ ] Sao Mai staff trained on troubleshooting
- [ ] At least 2 students have successfully used the lab
- [ ] All documentation handed over to Sao Mai
- [ ] Follow-up support schedule confirmed
- [ ] Contact information exchanged
- [ ] Photos/videos taken for documentation (with permission)

## Post-Deployment

### Week 1 Follow-up
- [ ] Daily check-in with Sao Mai (Days 1-7)
- [ ] Document any issues reported
- [ ] Provide remote support as needed
- [ ] Collect initial feedback from students

### Month 1 Review
- [ ] Review usage statistics (if available)
- [ ] Assess student progress
- [ ] Identify any recurring issues
- [ ] Plan any necessary adjustments

### 3-Month Evaluation
- [ ] Measure against success metrics
- [ ] Gather testimonials from students/staff
- [ ] Document lessons learned
- [ ] Plan for sustainability and expansion
